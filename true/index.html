Getting Line of Thought Working in IE

No one likes getting a layout working in Internet Explorer. Luckily there has been a major movement in the web world to not support older versions of IE as usage of those browsers weens. That’s the aim we’re taking with Line of Thought as well, and plan to only test the site in the current version and previous version of major browsers. Now that IE 9 is out, that means getting this blog and Line of Thought working in only IE 8 and 9. One morning this week I switched over to my girlfriends old windows computer to see the horror.

HTML 5 and Internet Explorer 8

The first browser tried was IE 8. Since Line of Thought uses a number of HTML 5 tags including <header>, <nav>, <section>, <aside> and <footer>, it was pretty obvious the site was going to fail spectacularly, and sure enough it did.

Luckily there’s a great workaround for this in the form of a shim script. You can read more about this HTML5 enabling script or give it a try. Just make sure to include this in the <head> of your document.

Liquid error: No such file or directory – pygmentize -l javascript -f html -O encoding=utf-8

So this script won’t even be included if you’re running IE 9 or later, but for IE 8 it’ll help. No css declarations had to change for this, it just worked.

One additional tag I somehow put in was a <sidebar> tag. Apparently I thought this was another new HTML 5 tag, but it’s not in the specification. Firefox and Chrome didn’t complain about this, so it worked fine and was styleable, but the Javascript code wouldn’t work with it. After going threw and changing these to <section class=“sidebar”> tags and changing a few styles everything was working fine.

Surprisingly that was the entirety of getting the site to work in IE8. No individual styles were added, no special cases or hacks required. Part of that is due to the fact that the layout is sparse, and doesn’t take advantage of many new effects. Those that it does use, such as gradients in buttons, degrade down to single color buttons which work fine in IE8. Luckily things don’t need to look identical. Total time to get the site working in IE8: less than 20 minutes.

Internet Explorer 9

IE 9 brings about support for most new HTML tags, including all the ones mentioned above.

What is Line Of Thought?

If you’re a passionate developer, chances are you enjoy talking to other people about the programming languages, frameworks, databases and other tools you’re using. At a basic level, just about every website out there divulges basic information about their server stack, be it a little Python here, or a little PHP there. Line Of Thought helps developers share this information with the world while aggregating it with everyone else who wants to share.

Our goal is to make it easy to create a list of tools used on any website. For instance, on the listing page for Line Of Thought we share just about all technology in our stack. It lists out all the basics you’d imagine (language, database, framework), but goes deeper into back end libraries. Which Ruby gems and Javascript libraries a site is using won’t be interesting to everyone, but if you’re looking to share with your peers, or looking for new tools to check out, this is a great way to do it fast.

Site Owners Looking to Share

If you’re a site owner, you can “claim” your site on Line Of Thought, then have full access to it. This allows you to edit any part of it, and lock it from being edited by the public. You can also see each revision of how a site is using an individual tool to see how it’s changed over time.

The biggest draw for site owners is just the ability to share your work. Rather than making a blog post listing out your technology stack, you could or answering questions on Quora and other Q&A sites, by entering your tools into Line Of Thought, there’s a definite location for this knowledge that everyone can refer to.

Tool Builders

So many times I’ve been on GitHub and seen a page detailing the sites that are using that project. This kind of page begs to be entered into Line Of Thought. Rather than doing this in a wiki format, or controlled by a single person, it can be handled on a site by site basis by those using it.

Accidental Discovery

Browsing Wikipedia is addicting. You start on one page and before you know it, you have 4 other tabs open for other subjects that looked interesting. Line Of Thought can be the same way. You might start by wondering which sites are using MongoDB and before long you’re going through the list of other Datastores to see what other sites are using.

Easy to Share

The share links on the side of each site and tool make it easy to share a page on Twitter or Facebook. If you share a site, it’ll grab random tools used on that site to share (of course you can enter your own as well). This helps make sharing a bit more dynamic. Refreshing the Facebook page a few times, the share message shows up as the following:

  • Facebook is using at least 19 tools including Apache Hive, MySQL and C (via Line Of Thought)
  • Facebook is using at least 19 tools including XHP, Waitr and PHP (via Line Of Thought)
  • Facebook is using at least 19 tools including Erlang, XHP and Facebook Scribe (via Line Of Thought)
  • Facebook is using at least 19 tools including Apache HBase, JSSpec and XHP (via Line Of Thought)

I’m way too biased here, but seeing a cool description like that I’d almost surely go check it out. :)
